Программа принимает список из любых элементов. В данном массиве необходимо найти первую самую длинную строку. В найденной строке необходимо найти первую самую длинную подстроку без повторений. Результат функции: индекс самой длинной строки списка и самая длинная подстрока без повторений. Если в исходном списке нет строк, то программа возвращает индекс -1 и пустую строку.
