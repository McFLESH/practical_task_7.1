Программа принимает список из любых элементов. Находит первую самую длинную строку. В ней находит первую самую длинную подстроку без повторений. Результат функции: индекс самой длинной строки списка и самая длинная подстрока без повторений. Если в исходном списке нет строк, то программа возвращает индекс -1 и пустую строку.
